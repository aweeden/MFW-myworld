# 🌍 MyWorld

A private, single-user social universe. Create profiles, post, comment, react, poll, message, form connections, build clubs with meeting chats and quizzes — all under your complete control.

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

Connect this repo to [Vercel](https://vercel.com) and it will auto-deploy on every push.

**Framework:** Vite  
**Build command:** `npm run build`  
**Output directory:** `dist`
